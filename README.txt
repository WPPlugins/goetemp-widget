== Description ==

With this plugin, the hourly current data of the weather station Nohlstraße in Göttingen be retrieved and displayed.

The data comes via the API of the page http://www.goetemp.de.

The following data can be displayed optionally:

* fine dust
* nitrogen dioxide
* sulfur dioxide
* temperature
* air pressure
* wind direction
* wind speed
* humidity
* rainfall duration
* global radiation
* UV-Index
* ozone
* last update

In the WordPress backend, these values can be enabled, or disabled. The title of the widget can be changed as well.

== Installation ==

You can download and install Goetemp using the built in WordPress plugin installer. If you download Goetemp manually, make sure it is uploaded to "/wp-content/plugins/goetemp-widget/".
